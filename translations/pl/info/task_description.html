Znajdź wartość najbliższą do podanej liczby.

Dana jest lista wartości w formie {% if interpreter.slug == "js-node" %}tablicy (Array){% else %}zbioru (set){% endif %} oraz liczba, dla której należy znaleźć najbliższą wartość.

Weźmy przykładowy zbiór liczb: 4, 7, 10, 11, 12, 17, oraz liczbę 9, dla której należy znaleźć najbliższą wartość. Jeśli posortujemy zbiór rosnąco, wtedy po lewej stronie liczby 9 będzie 7, a po prawej - 10. Natomiast 10 jest bliżej niż 7 i dlatego poprawną odpowiedzią jest liczba 10.

Dla doprecyzowania:

  • Jeżeli dwie liczby są w takiej samej odległości, należy wybrać mniejszą z nich;
  • Zbiór liczb jest zawsze niepusty, tzn. jego wielkość >=1;
  • Zbiór może zawierać zadaną liczbę, co oznacza, że jest ona zarazem rozwiązaniem;
  • Zbiór może zawierać liczby dodatnie i ujemne, lecz zawsze są to liczby całkowite;
  • Zbiór nie jest posortowany, a liczby się nie powtarzają.

Wejście: Dwa argumenty. Set zawierający liczby oraz wartość szukana typu int.

Wyjście: Int.
